# 麻雀のメンバーをいい感じに分けるやつ
![スクリーンショット](https://raw.githubusercontent.com/kousokujin/janshi_Separate/master/img/img.png)
## 使い方
左側に参加するメンバーの名前を入力して分けるボタンを押すだけ。

メンバーは改行で分けます。

## Seed値
乱数のSeed値を決められます。同じSeed値だと常に同じ結果が出ます。

Seed値は数字でも文字でもいけます。
